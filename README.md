# UfindAI

UfindAI is a platform for discovering and exploring AI tools across multiple categories.

## Features
- AI tool discovery
- Categories and filtering
- Admin dashboard
- Dark/light mode
- Responsive UI

## Tech Stack
- React
- Vite
- Tailwind CSS
- DaisyUI
- Node.js / Express
- MongoDB
